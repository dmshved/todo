# ToDo

Simple Todo Application for **[SoftPlus](https://softplusua.com/)** :)

# Prerequisites
- 🐳 Docker

# Painless installation

1. Clone the repo

```
https://github.com/dmshved/todo.git
```

2. Run the application

```
docker compose up -d
```

3. Access API documentation

```
http://localhost:5000/scalar/
```
