### ToDo

Simple Todo Application for **[SoftPlus](https://softplusua.com/)** :)

### Prerequisites
- 🐳 Docker

### Installation

1. Clone the repo

```
git clone https://github.com/dmshved/todo.git && cd todo
```

2. Run the application

```
docker compose up -d
```

3. Access API documentation

```
http://localhost:5000/scalar/
```

4. Login with default credentials to get the JWT access token (valid 15 minutes)

```
email: user@test.com
password: Password12345!
```
